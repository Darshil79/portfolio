File Structure:
* index.html
    * Purpose: Main landing page introducing Darshil and showcasing a welcome message.
    * Dependencies: StyleSheet.css
        * Additional Information:
        * Header and navigation are styled for a dark theme.
        * Main body divided into mainbodyleft and mainbodyright sections.
* aboutME.html
    * Purpose: Provides a detailed background, story, and skills overview of Darshil.
    * Dependencies: StyleSheet.css
        * Additional Information:
        * Two-column layout (secondSectionLeft and secondSectionRight).
        * Profile image displayed on the left with accompanying text on the right.
* services.html
    * Purpose: Highlights the services offered by Darshil.
    * Dependencies: StyleSheet.css
        * Additional Information:
        * Services presented in a grid layout (services-container).
        * Each service card contains an image, title, and description.
* projects.html
    * Purpose: Showcases notable projects with title, description, and view project link.
    * Dependencies: StyleSheet.css
        * Additional Information:
        * Project cards arranged in a grid layout (projects-container).
        * Hover effect implemented for visual feedback on project cards.
* contactME.html
    * Purpose: Facilitates contacting Darshil through a contact form.
    * Dependencies: StyleSheet.css
        * Additional Information:
        * Form inputs styled for a cohesive appearance (contact-form).
        * Submit button with a hover effect for an intuitive user experience.
* StyleSheet.css
    * Purpose: Defines visual styling for all HTML pages.
    * Dependencies: None
        * Additional Information:
        * Includes styling for the header, navigation, main body sections, service cards, project cards, and contact form.
        * Dark background theme with yellow accents for a professional and modern look.
